# Minito

실습실 컴퓨터 모니터링 및 제어 시스템입니다.

---

## 프로그램 개요
* **교수자용**: 실습실 PC 화면을 모니터링하고, 수업과 관련없는 프로세스를 강제 종료할 수 있는 대시보드 앱입니다.
* **학생용**: 백그라운드에서 동작하며, 교수자용 대시보드로 화면 캡처 본과 프로세스 리스트를 주기적으로 송출하는 에이전트 앱입니다.

---

## 실행 및 연동 방법

### 1. 프로그램 다운로드
* 저장소의 [GitHub Releases](https://github.com/JH23e/Minito/releases) 탭에서 최신 빌드된 인스톨러 바이너리를 다운로드합니다.
  * 교수용 PC: `Minito_professor_setup.exe`
  * 학생용 PC: `Minito_student_setup.exe`

### 2. 학생용 PC 설정 (에이전트)
1. 학생용 PC에서 `Minito_student_setup.exe`를 실행하여 설치를 완료합니다.
2. 설치 완료 시 나타나는 설정 대화창에서 **해당하는 실습실(예: AI융합실습실 4실)**과 **PC 번호**를 지정하고 완료를 누릅니다.
3. 프로그램은 백그라운드 서비스에 등록되어 부팅 시마다 자동 실행됩니다.

### 3. 교수자용 PC 설정 (대시보드 관제)
1. 교수용 PC에서 `Minito_professor_setup.exe`를 실행하여 설치합니다.
2. 최초 실행 시 나타나는 설정 창에서 **해당하는 실습실(예: AI융합실습실 4실)**을 선택합니다.
3. 대시보드가 실행되면 동일한 실습실 번호로 지정된 학생 PC들이 **감지되어 모니터링이 시작**됩니다.
